alphacolor

alphacolor — RGBA to AYUV colorspace conversion preserving the alpha channel

Synopsis




                    GstAlphaColor;

Object Hierarchy


  GObject
   +----GstObject
         +----GstElement
               +----GstBaseTransform
                     +----GstAlphaColor

Description

The alphacolor element does memory-efficient (in-place) colourspace conversion from RGBA to AYUV, preserving the alpha channel.
