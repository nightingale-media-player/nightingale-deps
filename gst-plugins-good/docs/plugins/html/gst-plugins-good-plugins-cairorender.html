cairorender

cairorender — Encodes streams using Cairo

Synopsis

                    GstCairoRender;

Object Hierarchy

  GObject
   +----GstObject
         +----GstElement
               +----GstCairoRender

Description

cairorender encodes a video stream into PDF, SVG, PNG or Postscript

Example launch line

gst-launch videotestsrc num-buffers=3 ! cairorender ! "application/pdf" ! filesink location=test.pdf
