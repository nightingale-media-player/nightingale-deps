mimenc — Mimic encoder

Synopsis

                    GstMimEnc;

Object Hierarchy

  GObject
   +----GstObject
         +----GstElement
               +----GstMimEnc

Properties

  "paused-mode"              gboolean              : Read / Write

Description

The MIMIC codec is used by MSN Messenger's webcam support. It creates the TCP header for the MIMIC codec.

Its fourcc is ML20.
