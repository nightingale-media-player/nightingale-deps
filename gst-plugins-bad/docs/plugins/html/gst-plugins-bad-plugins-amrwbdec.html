amrwbdec

amrwbdec — Adaptive Multi-Rate Wideband audio decoder

Synopsis




                    GstAmrwbDec;

Object Hierarchy


  GObject
   +----GstObject

Description

This is an AMR wideband decoder based on the reference codec implementation.

Example launch line

gst-launch filesrc location=abc.amr ! amrwbparse ! amrwbdec ! audioresample ! audioconvert ! alsasink
