alsasink

alsasink — Output to a sound card via ALSA

Synopsis

                    GstAlsaSink;

Object Hierarchy

  GObject
   +----GstObject
         +----GstElement
               +----GstBaseSink
                     +----GstBaseAudioSink
                           +----GstAudioSink
                                 +----GstAlsaSink

Implemented Interfaces

GstAlsaSink implements GstPropertyProbe.

Properties

  "device"                   gchar*                : Read / Write
  "device-name"              gchar*                : Read

Description

This element renders raw audio samples using the ALSA api.

Example pipelines

gst-launch -v filesrc location=sine.ogg ! oggdemux ! vorbisdec ! audioconvert ! audioresample ! alsasink
Play an Ogg/Vorbis file.
