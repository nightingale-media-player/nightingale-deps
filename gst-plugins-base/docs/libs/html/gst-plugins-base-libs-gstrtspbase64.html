Details

gst_rtsp_base64_encode ()

gchar*              gst_rtsp_base64_encode              (const gchar *data,
                                                         gsize len);

Encode a sequence of binary data into its Base-64 stringified representation.

data : the binary data to encode
len : the length of data
Returns : a newly allocated, zero-terminated Base-64 encoded string representing data.

gst_rtsp_base64_decode_ip ()

void                gst_rtsp_base64_decode_ip           (gchar *data,
                                                         gsize *len);

Decode the base64 string pointed to by data in-place. When len is not NULL it will contain the length of the decoded data.

data : the base64 encoded data
len : location for output length or NULL
