Description

Provides methods for creating and parsing request, response and data messages.

Last reviewed on 2007-07-25 (0.10.14)

Details

enum GstRTSPMsgType

typedef enum
{
  GST_RTSP_MESSAGE_INVALID,
  GST_RTSP_MESSAGE_REQUEST,
  GST_RTSP_MESSAGE_RESPONSE,
  GST_RTSP_MESSAGE_DATA,
} GstRTSPMsgType;

The type of a message.

GST_RTSP_MESSAGE_INVALID invalid message type
GST_RTSP_MESSAGE_REQUEST request message
GST_RTSP_MESSAGE_RESPONSE response message
GST_RTSP_MESSAGE_DATA data message

GstRTSPMessage

typedef struct {
  GstRTSPMsgType    type;

  union {
    struct {
      GstRTSPMethod      method;
      gchar             *uri;
      GstRTSPVersion     version;
    } request;
    struct {
      GstRTSPStatusCode  code;
      gchar             *reason;
      GstRTSPVersion     version;
    } response;
    struct {
      guint8             channel;
    } data;
} GstRTSPMessage;

An RTSP message containing request, response or data messages. Depending on the type, the appropriate structure may be accessed.

GstRTSPMsgType type; the message type
