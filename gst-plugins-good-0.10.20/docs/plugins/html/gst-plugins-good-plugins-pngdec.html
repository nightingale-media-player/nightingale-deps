pngdec

pngdec — Decode a png video frame to a raw image

Synopsis

                    GstPngDec;

Object Hierarchy

  GObject
   +----GstObject
         +----GstElement
               +----GstPngDec

Description

Decodes png images. If there is no framerate set on sink caps, it sends EOS after the first picture.
